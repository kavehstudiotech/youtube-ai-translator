# مترجم فارسی یوتیوب 🎬 — Persian YouTube Translator

افزونه‌ی کروم (Manifest V3) که زیرنویس ویدئوهای یوتیوب را به‌صورت خودکار می‌گیرد، با موتور **OpenRouter** به فارسی ترجمه می‌کند و **هم‌زمان روی ویدئو** نمایش می‌دهد.

## ویژگی‌ها
- ترجمه‌ی خودکار زیرنویس هر ویدئوی یوتیوب به فارسی
- نمایش زیرنویس روی ویدئو، سازگار با حالت تمام‌صفحه
- نمایش اختیاری متن اصلی (دوزبانه) برای زبان‌آموزی
- شخصی‌سازی کامل: فونت، رنگ، اندازه، ضخامت، رنگ و شفافیت پس‌زمینه، فاصله از پایین
- پیش‌نمایش زنده در پنجره‌ی تنظیمات
- انتخاب آزاد مدل OpenRouter (پیش‌فرض: `google/gemini-2.0-flash-001`)
- کش ترجمه برای سرعت بالا هنگام جلو/عقب بردن ویدئو

## نصب (حالت توسعه‌دهنده)
1. کروم را باز کنید و به `chrome://extensions` بروید.
2. گزینه‌ی **Developer mode** (بالا-راست) را روشن کنید.
3. روی **Load unpacked** بزنید و پوشه‌ی این پروژه را انتخاب کنید.
4. روی آیکون افزونه کلیک کنید و **کلید OpenRouter** خود را وارد کنید
   (از <https://openrouter.ai/keys> رایگان بگیرید) و «بررسی کلید» را بزنید.
5. یک ویدئوی یوتیوب باز کنید — زیرنویس فارسی خودکار ظاهر می‌شود.

## ساختار پروژه
```
manifest.json          ← تعریف افزونه (MV3)
src/
  inject-main.js        ← پل در دنیای MAIN: استخراج لیست زیرنویس‌ها از پلیر
  content.js            ← هماهنگ‌کننده: دانلود زیرنویس، ترجمه، نمایش هم‌زمان
  background.js         ← Service worker: ترجمه‌ی دسته‌ای با OpenRouter + کش
  subtitle.css          ← استایل نوار زیرنویس و اعلان‌ها
popup/                  ← پنجره‌ی تنظیمات (UI شخصی‌سازی + پیش‌نمایش زنده)
icons/                  ← آیکون‌های افزونه
```

## چطور کار می‌کند
1. `inject-main.js` در دنیای MAIN صفحه اجرا می‌شود و از `#movie_player` لیست
   مسیرهای زیرنویس (`captionTracks`) را می‌گیرد و به اسکریپت ایزوله پاس می‌دهد.
2. `content.js` بهترین مسیر (ترجیحاً انگلیسی دستی، بعد خودکار) را با فرمت
   `json3` دانلود و به cue تبدیل می‌کند.
3. cueها به‌صورت دسته‌های ۲۰تایی به `background.js` فرستاده و با OpenRouter به
   فارسی ترجمه می‌شوند؛ نتیجه کش می‌شود.
4. یک حلقه‌ی `requestAnimationFrame` زمان فعلی ویدئو را با cueها هم‌زمان می‌کند
   و نوار زیرنویس را نمایش می‌دهد.

## نکات و محدودیت‌ها
- کلید OpenRouter فقط به‌صورت محلی در `chrome.storage.sync` ذخیره می‌شود و فقط
  به سرور OpenRouter ارسال می‌گردد.
- اگر ویدئویی هیچ زیرنویسی (دستی یا خودکار) نداشته باشد، چیزی برای ترجمه نیست.
- یوتیوب گاهی برای دریافت زیرنویس محدودیت اعمال می‌کند؛ در این صورت صفحه را
  تازه کنید.
- هزینه‌ی ترجمه بر اساس مدل انتخابی شما در OpenRouter محاسبه می‌شود
  (مدل‌های flash بسیار ارزان‌اند).
